 Shopping List App

A beautiful and functional shopping list application built with React Native, Expo, and Redux Toolkit.

# Features

- *Complete CRUD Operations**
  - Create new items (custom + quick add)
  - Read/View shopping list with images
  - Update existing items
  - Delete items

- Smart Categories
  - Fruits 
  - Vegetables  
  - Dairy & Eggs 
  - Pantry Items 

- *Real Shopping List Experience*
  - Quantity and unit tracking
  - Mark items as purchased
  - Progress tracking (Total/Done/Left)
  - Price estimates for common items

- ** Persistent Storage**
  - Automatically saves your list
  - Works offline
  - Data persists between app restarts

- **Beautiful UI**
  - Clean, intuitive interface
  - High-quality product images
  - Color-coded categories
  - Responsive design
